# 项目创建

本地：
- @vue/cli 4.5.14

项目：
- Vue2.x
- Babel
- Router


# 项目初始化

配置 npm ，Scripts 使用 serve

安装依赖：
```
npm install axios@0.26.0
npm i element-ui -S
```
